      length  获取数组的长度

      功能：去掉数组中的第一个元素

      功能：向数组的末尾添加新的元素

      添加的数组元素
      返回：被添加元素后的数组的长度

      数组对象.unshift(数组元素)  
      功能：向数组的头添加新的元素
      参数：

      返回：被添加元素后的数组的长度

      数组对象.sort()
      功能：对数组对象中的元素进行排序
      返回：排序后的数组

四、JavaScript的自定义函数
       自定义函数可以重复执行的代码段。
    
       自定义函数最多可有以下4个组成部分
       - 函数名称
       - 参数（可选）
       - 函数体
       - 返回值（可选）

       1、自定义函数的声明方式
             - function 函数名称([参数...]){
	            函数体
                }
               函数名称 数字、字母、下划线，数字不能作为函数名称开头
                              函数名称要尽量起得有意义
                              函数名称尽量采用小驼峰式命名规则

            - var 变量 = new Function([string 参数,...,string函数体])

            - 匿名函数
               var 变量 = function(){
                      函数体
                }          

      2、自定义函数的调用
            - 以代码形式直接在JavaScript代码段中调用自定义函数的名称             - 以超链接形式调用自定义函数
              <a href="javascipt:自定义函数的名称"></a> 
            - 事件触发


五、JavaScript DOM
       1、DOM 
             Document Object Model(文档对象模型)
             Document  HTML/XML 
             
             XML eXtensible  Markup Language(可扩展标记语言)
             XML没有约定好的标签。要求所有的标签必须成对出现。
             XML用途：记录和传递信息。 

            Object  要将HTML转换为JavaScript对象。

            Model 模型

        2、将HTML文档转换为JavaScript对象
              - document.getElementsByTagName(HTML Tag名称);
              - document.getElementsByName(HTML 元素的name属性值)
              - document.getElementById(HTML 元素的ID属性值)

        3、属性
              属性的读取：对象.属性名称
              属性的设置：对象.属性名称 = 属性值

        4、文本
              对象.innerHTML
              对象.innerText 
              区别：innerHTML是解析HTML标签的。innerText是不解析HTML标签的。

        5、样式
              样式的获取
              对象.style.样式名称
              注意：样式名称中有 -,将 - 去掉，然后下一个单词首字母大写。
              样式的设置
              对象.style.样式名称 = 样式值
